Painless order intake

The cashier uses a layout optimized for touch registers. Split orders, merge tables, apply promotions, and take UPI payment details. It stores tickets inside Supabase even if the connection goes offline momentarily.

Kitchen Ticket Dispatch

The system generates real-time chef tickets. Prep timers, item ingredients, modifiers (e.g. "Less Spicy"), and dining preferences are color-coded. Chefs bump finished dishes to clear the boards.

Stock Formula deduction

RestroSuite links menu recipes to raw stock categories. As bills settle, ingredients deduct. Receive instant alerts on your dashboard when stock crosses safety limits so you order raw material on time.

Free Notification receipts

Leverage the WhatsApp Gateway API integration. Bills convert to responsive HTML sheets and send to customer mobiles. Safe, environment-friendly, and helps retain guest visit records for analytics.

Frequently Asked Questions

Everything you need to know about the self-hosted zero-cost SaaS model.

How is RestroSuite completely free?

RestroSuite is open-source. Instead of paying a software provider, you host the code on a free-tier Vercel account and connect it to a free-tier Supabase database. These free tiers provide generous limits that easily cover standard restaurant volumes, meaning your software running cost is exactly zero.

Will it work if the internet goes offline?

Yes. The POS billing interface works in your browser cache. If the internet fails, you can continue taking orders and generating receipts. Once connection is restored, the local orders are synced back to the Supabase cloud automatically.

What hardware is required for counters and kitchens?

RestroSuite runs inside standard web browsers. You can use any existing hardware: iPads, Android tablets, old laptops, or dedicated touch POS terminals. You do not need to buy any proprietary hardware to run our POS.

How does the WhatsApp Receipt system work?

We integrate with a local WhatsApp Web gateway script (included in the package) or external APIs. When an order is completed, the receipt converts to a neat digital message and dispatches automatically. This eliminates the need for expensive paper thermal printers.
